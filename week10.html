Pointers are variables; thus, a pointer also has a value.
Pointers are the variables that store memory addresses. Usually, pointer stores a value, which is an address , say variable.

Variable Pointer
Stores value directly
Can Access the value directly		 Stores address of another variable.Can access the value indirectly (access the address of variable directly, and access its value)
Referencing a value using pointers is called indirection.

Pointers - declaraton

int *aPtr,*ptrA,*pA;
float *bPtr,*ptrB,*pB;
char *cPtr,*ptrC,*pC;

Just like any variable, pointers must be declared before they are used:
type *pointer_name;
Note:Note: asterisk (*) is a prefix, it does not apply to all the variables in the declaration.
int *aPtr;
aPtr is a variable.
aPtr is also a pointer to an int.
Note: later you will understand better why it is important to state pointer to int, and how it is different than, say, pointer to char.

Pointers assigment

Pointers, like any other variable can be declared and assigned some value at the same time (initialization) or later.

int a = 10;
int *aPtr;
aPtr = &a;
& is unary address of operator. 
int *aP=aPtr;

Header defines symbolic constant NULL which can be used to initialize pointers, if you are not sure of the initial value. ( also defines it) NULL indicates that the pointer is pointing to NOWHERE currently. An alternative is to use 0(ZERO) (but it is not recommended).

Function call by reference

When you pass an argument by value, all the changes made within the function will lose affect on the original value. When you pass an argument by reference though, the modifications are made directly on the original value. Using address of (&) and indirection (*) operators, any variable can be passed into a function by reference.
